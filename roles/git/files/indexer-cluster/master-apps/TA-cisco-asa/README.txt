The Splunk Add-on for Cisco ASA allows a Splunk Enterprise administrator to collect Cisco ASA, VPN, FWSM, and/or PIX system logs. You can use Splunk Enterprise to analyze these logs directly or use them as a contextual data source to correlate with other communication and authentication data in Splunk Enterprise. This add-on provides the inputs and CIM-compatible knowledge to use with other Splunk Enterprise apps, such as Splunk App for Enterprise Security and Splunk App for PCI Compliance.
