##################
##   Splunk     ##
##################

Basic installation of Splunk

# Dependencies

- splunklib-1.6.6

  Required for the custom Ansible modules to manage the Splunk.

# Tasks

- Set hostname

- Add Splunk user

- Unpack Splunk tarball into Splunk Home(/opt/splunk)

- Start Splunk and Accept License

- Enable boot-start

- Change default password
